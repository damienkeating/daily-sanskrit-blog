# Daily Sanskrit Learning

A multilingual blog that explores historical events through the lens of Sanskrit vocabulary.

Each day, a random year is chosen and a significant historical event from that date is summarized in:
- **Sanskrit** (with Devanagari script)
- **English**
- **Chinese**

Each post includes 3 thematically-linked Sanskrit vocabulary words with transliteration and explanations.

## Live Site

https://damienkeating.github.io/daily-sanskrit-blog/

## Structure

- `_posts/` — Daily Markdown posts (Jekyll format)
- `_layouts/` — HTML templates
- `assets/css/` — Stylesheets

## How It Works

This blog is automatically generated and published daily by [Hermes Agent](https://github.com/hermes-agent).

Posts are sourced from Wikipedia historical events, with Sanskrit content generated via AI.
